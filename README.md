# Pixel Fusion Portfolio

This repository contains a personal portfolio website built with Vite, React, TypeScript, Tailwind CSS and shadcn-ui.

## Project URL

- Repository: https://github.com/dhivya-shreetha-s/portfolio

## Development

Requirements: Node.js (recommended via nvm), and a package manager (npm, pnpm or bun).

Quick start:

```sh
# Clone the repository
git clone https://github.com/dhivya-shreetha-s/portfolio.git
cd portfolio

# Install dependencies (example using npm)
npm install

# Start development server
npm run dev
```

## Technologies

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

## Notes

- Open `src/pages/Contact.tsx` and `src/pages/SinglePage.tsx` to configure EmailJS credentials if you use EmailJS for contact form delivery.
- The `public/` directory stores static assets (images, favicon, resume PDF).
